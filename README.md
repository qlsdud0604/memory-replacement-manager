# 메모리 교체 매니저

## 기술 스택
* IDE
```
 - Eclipse
```
* Language
```
 - Java
```
</br>

## 프로젝트 목적
* 다양한 페이지 교체 알고리즘을 직접 구현해 봄으로써 동작 과정을 이해한다.

* 각 알고리즘의 페이지 교체 과정을 분석하여 특성을 파악한다.

* 구현한 프로그램을 통해 모의실험을 함으로써 각 알고리즘을 평가해 본다.
</br>

## 프로젝트 기능
* 입력받은 데이터에 대한 hit, fault, hit ratio, fault ratio, EAT 지수 출력

* 참조 문자열에 대한 프레임의 교체 과정을 간트 차트로 출력
</br>

## 샘플 데이터
```
number of frames : 3
number of inputs : 20
reference strings : 7 0 1 2 0 3 0 4 2 3 0 3 2 1 2 0 1 7 0 1
```
